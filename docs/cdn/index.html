Use daisyUI from CDN

How to use daisyUI from a CDN?

Tailwind Plugin Use CDN
Warning
CDN files are not recommended for production
because you can't purge unused styles and file size will be large.

You don’t need to install anything.
Just add one of these to the head tag of your HTML:

  • daisyUI 2.6.4 CSS file + Tailwind CSS 3.0 JS file:

<link href="https://cdn.jsdelivr.net/npm/daisyui@2.6.4/dist/full.css" rel="stylesheet" type="text/css" />
<script src="https://cdn.tailwindcss.com"></script>
Or
  • daisyUI 2.6.4 + Tailwind CSS 2.2 CSS files:

<link href="https://cdn.jsdelivr.net/npm/daisyui@2.6.4/dist/full.css" rel="stylesheet" type="text/css" />
<link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2/dist/tailwind.min.css" rel="stylesheet" type="text/css" />
